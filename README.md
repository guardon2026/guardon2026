# GuardOn

경비·보안 인력 긴급 매칭 플랫폼 (Korean B2B security staffing platform).

**Core value:** SOS 긴급 매칭 — resolve same-day staffing gaps in under 8 minutes.

## Stack
Next.js 14 App Router · TypeScript · Tailwind CSS · Prisma · PostgreSQL + PostGIS · next-auth v5 · shadcn/ui · Pretendard font.
